# 按钮布局更新说明

## 更新内容

### 右上角按钮组重新设计

将所有右上角功能按钮统一到一个垂直按钮组中，包括：

1. **🔒 锁定/解锁按钮**
   - 位置：右上角按钮组顶部
   - 样式：红色主题
   - 功能：锁定/解锁手机控制
   - 文字：🔒 已锁定 / 🔓 未锁定

2. **✅ 连接状态指示器**
   - 位置：锁定按钮下方
   - 样式：绿色徽章
   - 显示：仅在已连接时显示

3. **↺ 重置按钮**
   - 位置：连接状态下方（按钮组底部）
   - 样式：金色主题
   - 功能：重置所有抽奖数据
   - 文字：↺ 重置

### 样式统一

所有按钮使用统一样式类 `.btn-action`，具有：

- **统一尺寸**: `padding: 0.8em 2.5em`
- **统一圆角**: `border-radius: 1em`
- **统一样式**: 渐变背景、发光效果、悬停动画
- **统一交互**: `scale(1.05)` 悬停、`scale(0.95)` 点击
- **视觉一致性**: 相同的间距、边框、阴影

## 按钮组结构

```
┌─────────────────┐
│  🔒 已锁定     │  ← 锁定按钮（红色主题）
│                │
│  ✅ 已连接     │  ← 连接状态（绿色徽章）
│                │
│  ↺ 重置       │  ← 重置按钮（金色主题）
└─────────────────┘
```
